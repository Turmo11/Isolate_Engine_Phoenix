What's Isolate Engine Phoenix?

Isolate Engine Phoenix is a basic 3D Graphics Engine for game development forked from NULL ENGINE by Ángel González. It has been developed by a single Bachelor’s degree in Video Game Design and Development student at UPC CITM Tech Talent Center in Barcelona. The surname of phoenix, comes after having fallen in the previous deliveries. In this, the engine is reborn from its ashes to become the ultimate engine.

Who developed it?

Guillem Turmo (Turmo11)

Team brain and hard code programming.

Fran Guerrero (FranGV98)

Programmer in charge of the most visual part of the engine (UI, camera, basic geometry, etc.)

Main core sub-systems

  • Window configuration options
  • OpenGL configuration options
  • Render configuration options
  • Hierarchy & Inspector
  • GUI Docking
  • Basic Asset Explorer Window
  • GameObject & Components
  • Camera frustum culling
  • Unity-like camera movement
  • Objects have Bounding Boxes
  • Object transformation via Gizmos
  • Resource Management
  • Serialization
  • Scene Saving
  • Assets Window
  • Creation of custom files
  • All resources handled by their UIDs

    • Scripting system

    Import your Wren scripts to the engine for a quicker implementation without having to work with Engine Code

  • Wren Interpreter
  • Text Editor
  • Hot-Reloading (edit your scripts without having to recompile the engine)

    • NOTE: Some of this features might be limited or not finished
